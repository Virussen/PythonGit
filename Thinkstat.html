描述性统计量¶

均值和平均值¶

假设一个包含n个值的样本Xi,他们的均值u就等于他们的值的总和除以值的数量，这是非常显而易见的事情，如公式：

$$\mu =\frac{1}{n}\sum x_{i}$$

值得注意的是，在很多情况下均值(mean)和平均值(average)在很多情况下是相同的，但是在本质上依然是有区别的：

  • 样本的“均值”是根据公式计算出来的一个汇总统计量
  • 而“平均值”是若干种可以描述样本的典型值或者集中趋势的汇总统计量之一

方差¶

上面的例子，很明显的说明了南瓜的平均值并不能很好的概括一个数组的情况，为了让问题能够解决，我们引入了方差。他既是平均值的助手也是均值的对立。

  • 方差能够很好的涵盖上述南瓜的问题
  • 方差描述分散情况而非集中趋势，这正好与均值的集中趋势相反
$$\sigma ^{2}=\frac{1}{n}\sum (x_{i}- \mu )^{2}$$$$x_{i}- \mu$$

这一部分是离均差，即方差是偏差的方均值。取方差的算术根就是标准差。

相对于方差而言标准差的优势在于标准差的量纲是和统计量一致的。

def PercentileRank(scores, your_score):
    count = 0
    for score in scores:
        if score <= your_score:
            count += 1
    percentile_rank = 100.0 * count / len(scores)
    return percentile_rank

累积分布函数 CDF¶

CDF是什么？最简单的理解方式就是CDF函数就是值到其在分布中百分等级的映射。

CDF是关于x的函数，其中x是分布中的某一个值，计算给定x的CDF（x），其实就是计算小于等于x的值的比例。

In [19]:
def CDF(t, x):
    count = 0.0
    for value in t:
        if value <= x:
            count += 1.0
            
    prob = count / len(t)
    return prob

CDF与百分位数¶

事实上 CDF与上述的百分位数其实很相似，差别也仅仅是在于0~100和0~1而已。而0~1恰恰是柯尔莫哥洛夫提出的概率论三大公理之首。

概率密度函数¶

概率密度函数（PDF）最简单的解释便是CDF的一阶导数，因而相反的也可以说，累积分布函数（CDF）是概率密度函数的积分。

概率密度函数的值并非是概率，而是一种概率密度，类似于物理中我们所认识到的密度。即，概率密度衡量的是x轴上每个单位的概率。

指数分布¶

$$ y = 1 - e^{-\lambda x} $$$$P = \lambda e^{-\lambda x}$$

其中，参数λ决定了分布的形状。关于指数分布的特征值，我们有一下的结论：

$$E(X) = \frac{1}{\lambda }$$$$Median(X) = \frac{log(2)}{\lambda }$$$$Var(X) = \frac{1}{\lambda ^{2}}$$

正态分布¶

正态分布又称作高斯分布，由于其在自然界存在的广泛性，正态分布有着广泛的应用，尤其是随着中心极限定律的提出，正态分布的应用被提高到了前所未有的高度。

$$P(X;\mu,\sigma ) = \frac{1}{\sqrt{2\pi\sigma^{2}}}e^{-\frac{(x-\mu)^{2}}{2\sigma^{2}}}$$$$E(X)=\mu$$$$Var(X)=\sigma^{2}$$

正态分布的性质¶

$$ X \sim N(\mu,\delta)$$

那么对于X' = aX+b（a，b为实数）这样的线性变换来说，我们直接给出结论：

$$ X' \sim N(a\mu + b, a^{2}\delta)$$

对于多元情况来说，Z = X + Y，并且X, Y都服从于正态分布，那么我们也可以给出结论：

$$Z \sim N(\mu_{X}+\mu_{Y}, \delta_{X}^{2}+\delta_{Y}^{2})$$

中心极限定理概说¶

在上述的讨论中，我们已经知道，正态分布有着这样的性质：

  • 若将一些服从正态分布的数据加起来，和也将服从正态分布。

现在我们提出的是：如果将大量服从某种分布的值加起来，所得到的和会收敛到正态分布。

假设随机变量X的均值和标准差为μ何δ，那么n个随机变量X的和渐进的服从N(nμ，nδ²）

定理成立的条件：

  • 数据满足独立性
  • （可放宽的）数据必须服从同一个分布
  • E(X)与Var(X)必须是有限的/存在的
  • 收敛的速度取决于原分布的偏度

概率¶

在上述中，我们已经提到了概率求取的一种方式，即在有频数n与样本数量N的时候，我们采取$$ P = \frac{n}{N}$$ 作为概率。

我们假设各位已经有了概率论的基本认识，从而跳入概率认识的两大学派之中，即：

  • 频率学派（frequentism）
  • 贝叶斯学派（bayesianism）

频率学派我们已经很熟悉了，在大量实验的基础下，频率即为概率。而贝叶斯学派，则是根据贝叶斯公式而推倒出的整个概率论的学派。

$$P(B_{i}|A) = \frac{P(B_{i}P(A|B_{i}))}{\sum P(B_{j})P(A|B_{j})}$$

概率论的常用公式¶

  1. 在事件彼此独立的情况下：
$$P(\prod A_{i}) = \prod P(A_{i})$$
  1. 在不独立的情况下，通过条件概率公式我们可以获得：
$$P(AB) = P(A)P(A|B)$$

由二项分布提出的更有意义的问题——蒙特卡洛模拟¶

当所要求解的问题是某种事件出现的概率，或者是某个随机变量的期望值时，它们可以通过某种“试验”的方法，得到这种事件出现的频率，或者这个随机变数的平均值，并用它们作为问题的解。这就是蒙特卡罗方法的基本思想。蒙特卡罗方法通过抓住事物运动的几何数量和几何特征，利用数学方法来加以模拟，即进行一种数字模拟实验。它是以一个概率模型为基础，按照这个模型所描绘的过程，通过模拟实验的结果，作为问题的近似解。可以把蒙特卡罗解题归结为三个主要步骤：构造或描述概率过程；实现从已知概率分布抽样；建立各种估计量。

假设检验¶

假设检验的过程：

  • 原假设：基于假设的系统模型
  • p值：出现直观效应的概率
    • 通常，会为p选择一个阈值a，一旦p小于a，则推翻假设
    • 一般情况下，选择5%较多
  • 解释：基于概率推断观测是否具有统计的显著性

容易发生的错误¶

  • I类错误——假阳性。即接受了一个本质为假的假设。
  • II类错误——假阴性。即推翻了一个本质为真的假设。

改变阈值只能让此消彼减，同时解决的方法只有增加样本数量。

解释统计检验的结果¶

  • 古典方法：若p小于阈值a，则我们称为效应在统计学上是显著地，但不提出效应真实存在的结论。

  • 实际情况：上述的严谨解释用的非常少，这也算作，统计学被滥用的一个例子。

  • 贝叶斯统计解释：贝叶斯统计解释最简单的解释逻辑是，先验概率与后验概率的比较，后者高于前者则是有意义的

估计¶

从一个样本来推测总体的具体情况，便成为估计，用来估计分布参数的统计量，称为估计量。

主要流行的方法有：

  • 最小化均方误差 MSE
  • 极大似然估计量 MLE

估计量的偏性：

  • 有偏的
  • 无偏的

估计的类型：

  • 点估计： 用估计量产生一个值来估计参数
  • 置信区间：求取一个有界区间，从而能够覆盖未知的参数

相关性¶

  • 皮尔逊相关系数：将所有的值转化成标准分数 $$ Z = \frac{x-\mu}{\delta} $$
  • 斯皮尔曼相关系数：将所有的值转化成百分等级

协方差¶

协方差是用来衡量相关变量的变化趋势是否相同，不妨设有两序列X，Y，其均值离差为： $$ dx_{i} = x_{i}-\mu_{i}$$ $$ dy_{i} = y_{i}-\mu_{i}$$

而协方差就是这两结果积的均值：

$$Cov(X,Y) = \frac{1}{n}\sum dx_{i}dy_{i}$$
  • 注意，X，Y的维度必须是相同的。
  • 协方差的优点：易于计算
  • 协方差的缺点：不易解释，往往量纲是没有意义的。

相关性¶

首先，先要解决协方差的不利之处，即量纲无意义的问题，因而我们采用标准分数来计算。

$$ p_{i} = \frac{x_{i}-\mu_{X}}{\sigma _{X}}\frac{y_{i}-\mu_{Y}}{\sigma _{Y}}$$

于是可以算出乘积的均值，即皮尔逊相关系数：

$$ \rho = \frac{1}{n} \sum p_{i}$$

显然，我们能得到更容易计算的形式即：

$$ \rho = \frac{Cov(X,Y)}{\sigma_{X} \sigma_{Y}}$$$$ \rho = \frac{\sum dx_{i}dy{i}}{\sum dx_{i} \sum dy_{i}} $$

对于上式，利用可惜施瓦茨不等式即可证明 $$ -1 \leqslant \rho \leqslant 1 $$

最小二乘法¶

估计斜率最常用的方法便是最小二乘法，最早关于残差最小是由高斯提出的。

假设我们有一个数据序列 X，要通过 X 的一个函数来预测另一个数据 序列 Y。如果这个预测函数是线性的，截距为 α，斜率为 β，那么我们 可以预期 yi 大约会等于 α+βxi 。

除非两者是完全线性相关，否则势必会有残差出现，残差的公式为：

$$\varepsilon = (\alpha + \beta x_{i}) - y_{i}$$

形成残差可能的原因¶

  • 测量误差
  • 非随机因素

最小二乘法拟合的步骤¶

  1. 计算两序列x,y的均值，计算x的方差，x,y的协方差
  2. 估计斜率：
$$\widehat{\beta} = \frac{Cov(X,Y)}{Var(X)}$$

3.估计截距

$$\widehat{\alpha} = \overline{y} - \widehat{\beta}\overline{x}$$

拟合优度¶

我们可以计算模型的确定系数，来评价模型的预测能力：

$$R^{2} = 1- \frac{Var(\epsilon)}{Var(Y)}$$

而均方误差 $$MSE = Var(Y)$$
