# 数据分析纲要
## 简介
此类课程的主要内容将从理论与理论实现的两个方面来诠释数据分析的特性与行为，因此课程将主要包括的内容有：
* 数据分析理论
* Python语言
	* Python语言基础
	* Python与算法
	* Python的Module
	* Python与数据分析
	* Python与机器学习实现
* 数据库
	* SQLite
		* SQLite与Python的联合实现
	* MongoDB，由Python的dict引入
* 版本控制
	* Git
	* GitHub
